🔑 How to Fix the 'Something went wrong' Error (Get API Key)

Google Maps requires a special password called an API KEY to work. Follow these 5 simple steps:

  1. Go to the Cloud: Open the Google Cloud Console and sign in with your Gmail.
  2. Create a Project: Click 'Select a project' at the top and create a new one called My Fragrance Shop.
  3. Enable the Maps: Go to 'APIs & Services' -> 'Library'. Search for and click ENABLE on these three:
    • Maps JavaScript API (to show the map)
    • Places API (to search addresses)
    • Geocoding API (to turn the pin into text)
  4. Get the Key: Go to 'APIs & Services' -> 'Credentials'. Click + CREATE CREDENTIALS and choose 'API Key'.
  5. Paste the Key: Copy your new key and paste it into clinical files like index.html where it says YOUR_API_KEY.

Note: Google might ask for a credit card, but they give a huge free amount every month, so it's usually free for small sites!
